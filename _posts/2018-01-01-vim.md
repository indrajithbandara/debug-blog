### Dicas de comandos comum no vim
"/"  = nome_arquivo = busca arquivos (para passar proximo "n")
":!" = dรก qualquer comando como se tivesse no terminal
",e" = lista os arquivos no diretorio
",w ,q" = navega entre os arquivos
